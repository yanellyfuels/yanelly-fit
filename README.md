# Yanelly Fuels 🥑🍓

A simple, fast landing page for Yanelly Fuels — built to share simple nutrition education, repeatable meals, free guides, and Yanelly's RD2B journey.

---

## What's in this project

| File | What it is |
|------|-----------|
| `index.html` | The homepage text and sections. This is the main file to edit. |
| `styles.css` | The colors, fonts, layout, and visual design. |
| `script.js` | Mobile menu, reveal animations, email form success state, and footer year. |
| `images/yanelly.jpg` | The homepage photo. Replace this file to change the image. |
| `guides/` | Free guide preview image and downloadable PDF assets. |
| `.github/workflows/deploy.yml` | Auto-publishes the site through GitHub Pages when configured. |

---

## Quick edits

- To change homepage wording, edit `index.html`.
- To change the homepage photo, replace `images/yanelly.jpg` with the new photo using the same filename.
- To update socials, search for `instagram.com`, `tiktok.com`, or `youtube.com` in `index.html`.
- To add meals, copy one `<article class="recipe">` block in the meals section.
